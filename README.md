# Wohnvision Jurte

A sustainable yurt living project exploring alternative housing in Austria.

## About

Wohnvision Jurte documents the planning, legal research, and design process for establishing a permanent yurt dwelling in Austria. The project addresses housing affordability and ecological living through traditional Central Asian architecture adapted to European climate and building regulations.
